FILES
-----
1. index.html   - Home page
2. menu.html    - Filterable food and drinks menu
3. gallery.html - Responsive image gallery with JavaScript lightbox
4. about.html   - About Me page using the supplied profile photograph
5. contact.html - Contact and reservation form
6. styles.css   - Shared styling, responsiveness, themes and animations
7. script.js    - Theme switcher, scrolling effects, filters, lightbox and form logic
8. assets/profile.jpg - Profile photograph

HOW TO RUN
----------
1. Keep all files in the same folder structure.
2. Double-click index.html to open the website in a browser.
3. For the best development experience, open the folder in Visual Studio Code and use the Live Server extension.

IMPORTANT NOTES
---------------
- Food and restaurant photographs are loaded from Unsplash URLs, so an internet connection is needed for those images.
- The reservation form is a front-end demonstration. It validates the form and displays a success message, but it does not send data to a database or email service.
- The dark/light theme selection is remembered using localStorage.
